# Forensic Medicine Database UI

A React + Vite web interface for a Forensic Medicine Department database system.

## Features

- Landing page
- Login and role selection
- Role-based dashboards
- Patient registration
- Clinical case management
- Autopsy case management
- Evidence tracking
- Court reports and court dispatch
- Search interface
- Staff management
- Analytics and settings pages

## Running the project

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the local URL shown in the terminal, usually:

```text
http://localhost:5173
```

## Demo login

Use any username and password in the demo interface and choose a role from the login page.
